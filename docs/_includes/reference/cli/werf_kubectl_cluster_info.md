Display addresses of the control plane and services with label [kubernetes.io/cluster-service=true](kubernetes.io/cluster-service=true). To further debug and diagnose cluster problems, use &#39;kubectl cluster-info dump&#39;.

{{ header }} Syntax

```shell
werf kubectl cluster-info
```

{{ header }} Examples

```shell
  # Print the address of the control plane and cluster services
  kubectl cluster-info
```
